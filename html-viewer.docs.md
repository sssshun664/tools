# HTML Viewer

スマホからウェブページのHTMLソースコードを確認できるビューワー。イースターエッグや隠しメッセージを探すのに最適です。

## 機能

- **URLからHTML取得**: ウェブページのURLを入力してHTMLソースを取得
- **シンタックスハイライト**: Highlight.jsによる見やすいコード表示
- **HTMLコメント抽出**: ページに埋め込まれたHTMLコメントを一覧表示
- **テキスト検索**: HTMLソース内の特定のテキストを検索
- **統計情報**: 行数、文字数、コメント数を表示
- **コピー機能**: HTMLをクリップボードにコピー
- **ダウンロード機能**: HTMLファイルとしてダウンロード

## 使い方

### 基本的な使い方

1. URLを入力フィールドに貼り付ける
2. 「取得」ボタンをタップ
3. HTMLソースがシンタックスハイライト付きで表示されます

### HTMLコメントを探す

1. HTMLを取得後、「💬 コメント抽出」ボタンをタップ
2. ページ内のすべてのHTMLコメント（`<!-- -->`）が一覧表示されます
3. イースターエッグや開発者のメッセージを発見できます

### テキスト検索

1. 検索ボックスにキーワードを入力
2. 「検索」ボタンをタップ
3. マッチした行とその内容が表示されます（最大10件）

### その他の機能

- **📋 コピー**: HTMLソース全体をクリップボードにコピー
- **💾 ダウンロード**: HTMLファイルとして保存（`page.html`）

## ショートカット

- **URL入力でEnter**: HTMLを取得
- **検索ボックスでEnter**: 検索を実行

## 技術仕様

- **CORS対応**: AllOrigins APIを使用してCORS制限を回避
- **シンタックスハイライト**: Highlight.js (CDN)
- **対応ブラウザ**: モダンブラウザ全般（Chrome, Safari, Firefox, Edge）
- **ファイルサイズ**: 約10KB（外部ライブラリ除く）

## 使用例

### イースターエッグの探し方

1. 気になるウェブサイトのURLを入力
2. 「💬 コメント抽出」で隠しメッセージをチェック
3. 検索機能で特定のキーワード（例：`easter egg`, `secret`, `hidden`）を探す

### メタデータの確認

検索ボックスで以下を検索：
- `og:` - Open Graphメタタグ
- `twitter:` - Twitter Cardメタタグ
- `description` - ページ説明
- `author` - 作者情報

## 制限事項

- 一部のウェブサイトはCORSプロキシでも取得できない場合があります
- 非常に大きなHTMLファイルは表示に時間がかかる可能性があります
- JavaScriptで動的に生成されたコンテンツは含まれません（初期HTMLのみ）
- AllOrigins APIの利用制限に依存します

## プライバシーとセキュリティ

- HTMLの取得にはAllOrigins API（https://api.allorigins.win/）を経由します
- 機密情報を含むページのURLは入力しないでください
- 取得したHTMLはブラウザ内でのみ処理され、外部に保存されません
