# SVG Viewer

スマホ向けのシンプルなSVGビューワー。SVGコードをコピペしてその場でプレビューできます。

## 機能

- **リアルタイムプレビュー**: SVGコードを入力すると自動的にプレビューが更新されます
- **サンプルSVG**: 4種類のサンプルSVGをワンタップで読み込めます
  - 🔵 円: 同心円のデザイン
  - ⭐ 星: 星型のパス
  - 🌈 グラデーション: グラデーション付きの図形
  - ✏️ パス: ベジェ曲線のパス
- **エラー検証**: 無効なSVGコードの場合はエラーメッセージを表示
- **モバイル最適化**: スマホでの操作を考慮したUIデザイン

## 使い方

1. テキストエリアにSVGコードを貼り付ける
2. 「プレビュー」ボタンをタップ（または自動で表示）
3. プレビューエリアにSVGが表示されます

### ショートカット

- **Ctrl + Enter**: プレビューを更新
- **自動プレビュー**: 入力後0.5秒で自動的にプレビューが更新されます

## 技術仕様

- **ファイルサイズ**: 約8KB
- **依存関係**: なし（純粋なHTML/CSS/JavaScript）
- **対応ブラウザ**: モダンブラウザ全般（Chrome, Safari, Firefox, Edge）

## デザイン

- レスポンシブデザイン（モバイルファースト）
- グラデーション背景（#667eea → #764ba2）
- 最小タップターゲット: 48px
- カードベースのレイアウト

## 制限事項

- XSSを防ぐため、SVGコードはサニタイズされません（信頼できるソースからのみ使用してください）
- 外部ファイル参照（xlink:href等）は動作しない場合があります
- 非常に大きなSVGファイルはパフォーマンスに影響する可能性があります
